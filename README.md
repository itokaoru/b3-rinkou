# b3-rinkou
#### ローカルレポジトリ

+ 自分のパソコン内でバージョン管理するために作成したファイルやディレクトリの履歴を管理する場所のこと．

#### リモートレポジトリ

+ ネット上にファイルをアップロードした状態でファイル管理するもの．


#### コミット

+ ステージングエリアのファイルをローカルレポジトリに登録すること．

#### プッシュ

+ コミットをローカルレポジトリからリモートレポジトリに送る操作のこと．

#### フェッチ

+ リモートレポジトリの最新の履歴の取得だけを行うこと．

#### プル

+ リモートレポジトリからローカルレポジトリを更新する操作のこと．

#### ブランチ

+ 履歴の流れを分岐して記録していくためのもの．分岐したブランチは他のブランチは他のブランチの影響を受けない．

#### マージ

+ 分岐した履歴を戻して統合する手段のこと．
